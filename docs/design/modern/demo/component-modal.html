Modal Component Examples

Interactive demos of modal dialogs with animations and interactions

Modal Specifications

  • Maximum width: 600px
  • Padding: 32px
  • Border radius: 16px
  • Shadow: Level 3 (Floating)
  • Overlay: Semi-transparent black with backdrop blur
  • Animation: Fade in + scale from 0.95 to 1 + slide up (200ms)
  • Exit animation: Fade out + scale to 0.95 (150ms)
  • Interactions: Click overlay/X button to close, Escape key supported
  • Focus trap: Active when modal is open
